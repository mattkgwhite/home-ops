# Overview

## Usage

This is to understand how NixOS works, and how it can be used to configure specific things within NixOS itself via NixOS unique package management tools.

Usage for this is to deploy / experiment with NixOS as the base operating for configurations and then use NixOS' flake system to deploy, update and manage configuration of the machine.
